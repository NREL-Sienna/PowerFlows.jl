Solving a Power Flow

In this tutorial, you'll solve power flows on a 5-bus test system using three different solvers and compare their results.

Building a System

Create a System from PowerSystemCaseBuilder.jl:

julia> sys = with_logger(SimpleLogger(stderr, Logging.Error)) do
           build_system(MatpowerTestSystems, "matpower_case5_sys")
       end              System

DC Power Flow

Create a DCPowerFlow solver:

julia> pf_dc = DCPowerFlow()DCPowerFlow(nothing)

Solve the power flow with solve_powerflow:

julia> dc_results = with_logger(SimpleLogger(stderr, Logging.Error)) do
           solve_powerflow(pf_dc, sys)
       endDict{String, Dict{String, DataFrames.DataFrame}} with 1 entry:
